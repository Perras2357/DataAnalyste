# Projets d’Analyse de Données et Recherche d’Information

Ce dépôt regroupe **deux projets indépendants** réalisés dans un cadre académique, chacun implémenté dans son propre répertoire :

- **PageRank** : implémentation d’un moteur de recherche basé sur la popularité des documents.
- **ACP** : analyse exploratoire des données par Analyse en Composantes Principales.

Chaque projet est **autonome**, avec son propre pipeline, ses données et son README détaillé.
